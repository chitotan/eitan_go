## 概要
これは、あらかじめ登録しておいた単語と意味リストから問題を出題してくれたり、単語のフラッシュカードを表示してくれるなど、暗記勉強を効率よく行えるサイトテンプレートです。
本ファイルをダウンロードして使用することで、自分で問題を作成し、インターネットなしで利用することができます。

## 機能の説明

### 単語テスト(index.html)
・問題のランダム出題
・正誤判定
・点数の表示
・間違えた問題のみ再出題
・スクリーンショット

### 単語フラッシュカード
・問題のランダム表示・順番表示の切り替え
・全問題表示
・スクリーンショット



## ファイル構成
よく使用するファイル
/index.html → 単語テストをおこなうサイトです。
/card.html → 単語のフラッシュカードを表示するサイトです。
/create/index.html → 単語リスト(問題)生成サイトです。
/src/script/list.js →単語リストファイルです。

以下は編集しないファイル

/src/script/main.js → 単語テストをおこなうサイトのプログラムです。
/src/script/rightbutton.js → 画面右下のボタンのスクリプトです。
/src/css/style.css → 単語テストをおこなうサイトのCSSです。
/src/css/card-style.css → 単語のフラッシュカードを表示するサイトのCSSです。

/create/style.css →単語リスト(問題)生成サイトのCSSです。

## 使い方

### ①ダウンロードしたサイトにアクセスする。
本ファイルを利用するには、各ファイルをブラウザで開く必要があります。ここではWindowsを前提に解説します。

####単語テストサイトにアクセスする場合→
/index.html を右クリック→プログラムから開く→開きたいブラウザを選択

####単語フラッシュカードサイトにアクセスする場合→
/card.html を右クリック→プログラムから開く→開きたいブラウザを選択

※推奨:Google Chrome

### ②単語リストを作成する。
単語リストを作成するには、専用のツール「場所 : create/index.html」を使用することをお勧めします。
作成後、ファイルがダウンロードされるため、そのファイルを「src/script/list.js」に上書きしてください。
注)生成する前にタイトルは必ず入力し、ダウンロードされたファイル名は「list.js」であることを確認して、違う場合「list.js」にリネームしてください。
